# University Data Collection Checklist

A university can be marked as `complete` only when all required variables have
been researched and documented.

## Required factual variables

- [ ] Teaching language
- [ ] Estimated monthly cost
- [ ] Estimated housing cost
- [ ] Approximate return travel cost from Milan
- [ ] Visa requirement for EU citizens

## Required constructed scores

- [ ] Academic score
- [ ] Career score
- [ ] Lifestyle score
- [ ] International environment score
- [ ] Housing difficulty score
- [ ] English friendliness score

## Evidence requirements

- [ ] Every variable has an entry in `data/raw/data_sources.csv`
- [ ] Source type is recorded
- [ ] Confidence level is recorded
- [ ] Any project judgement is clearly identified
- [ ] Missing or uncertain information is documented

## Status definitions

### not_started

No meaningful research has been completed.

### partial

Some variables have been completed, but the university is not ready for final
comparison.

### complete

All required variables and source records have been completed.

### needs_review

The university was previously completed, but one or more sources or values may
be outdated.
